Funciones

Declaramos las variables en JavaScript con: var, let o const 

                var num1
                let num2
                const NUM3

Diferencias

Declaramos las variables en JavaScript con la palabra reservada function.

Las funciones tienen:

  1. Nombre.
  2. Parámetros.
  3. Resultado.
  4. Cuerpo.
                function nombre(parametro1, parametro2){
                    //Cuerpo con el código de la función.
                    return resultado
                }

Llamamos a las funciones por su nombre y dándole los parámetros. No olvidamos "recoger" el resultado.

Ejercicios de funciones

Estrategias para programar funciones

  1. Hacemos la DECLARACIÓN:
    • Elegir el nombre de la función.
    • Elegir el nombre de los parámetros.
    • Indentificado el tipo de dato del resultado.
  2. PROBAMOS la función:
    • Devolvemos un resultado "mock" (42).
    • Llamamos a la función sin parámetros y comprobamos que obtenemos el resultado mock.
  3. Buscamos TESTS (ejemplos).
  4. PROGRAMAMOS el cuerpode la función hasta que pasen los test.
  5. Si el problema es complicado, lo "simplificamos".
