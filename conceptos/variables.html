Declaración de variables

Declaramos las variables en JavaScript con: var, let o const 

                    var num1
                    let num2
                    const NUM3

Diferencias

var declara las variables al nivel "funcional". Es deci, el ámbito de la variable es toda la función que se declara.

Si estamos al máximo nivel, la variable es global.

Usando var la declaración pasa al principio de la función.

let declara las variables al nivel "de bloque" (dentro de llaves). Es decor, el ámbito de la variable es todo el bloque en el que se declara y sus bloques internos.

Si estamos al máximo nivel, la variable no "entra" en las funciones.

const declara las variables al nivel "funcional". Es decir, igual que var.

Moraleja: Si puedes utiliza const. Si no, usa let salvo que que necesites una variable global(usas var).
