Cómo se hace

Como se hace una clase:

                class NombreClase{}

Como se hace un objeto:

                let nombre_objeto = new NombreClase()

Como se hace un atributo:

                objeto.atributo = valor_atributo

Como se hace un método:

                objeto.metodo = function(){...}

this es la palabra reservada que hace referencia al objeto que ejecuta el código.

Relaciones entre clases

Uso

Una clase usa a otra clase cuando alguno de sus atributos o alguno de los parámetros o resultados de sus métodos son objetos de otra clase.

Ejemplo

                class Persona{}

                class Factura{

                    setCliente(cliente){ 
                        //Si cliente es un objeto de tipo Persona, entonces la clase Factura "usa" a Persona.

                        this.cliente = cliente
                    }
                }

Herencias

Es el mecanismo por el cual una clase permite heredar las características (atributos y métodos) de otra clase. La herencia permite que se puedan definir nuevas clases basadas de unas ya existentes a fin de reutilizar el código, generando así una jerarquía de clases dentro de una aplicación.

Super

El método super() es un uso especial de la palabra clave super donde se llama un constructor principal. En general, la palabra clave super se puede utilizar para llamar a métodos.

Ejemplo

                class Persona{

                    constructor(nombre){
                        this.nombre = nombre
                    }
                }

                class Factura extends Persona{

                    constructor(){

                        super() //LLama al constructor de la clase Persona
                    }
                }
