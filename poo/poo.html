Cómo se hace

Como se hace una clase:

            class NombreClase{}

Como se hace un objeto:

            let nombre_objeto = new NombreClase()

Como se hace un atributo:

            objeto.atributo = valor_atributo

Como se hace un método:

            objeto.metodo = function(){...}

this es la palabra reservada que hace referencia al objeto que ejecuta el código.

Relaciones entre clases

Uso

Una clase usa a otra clase cuando alguno de sus atributos o alguno de los parámetros o resultados de sus métodos son objetos de otra clase.

Ejemplo

            class Persona{}

            class Factura{

                setCliente(cliente){ 
                    //Si cliente es un objeto de tipo Persona, entonces la clase Factura "usa" a Persona.

                    this.cliente = cliente
                }
            }
